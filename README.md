# programme

This repository was reorganized to follow a common GitHub Python layout.

## Directory layout

- `src/`: main application source code
- `tests/`: test files
- `config/`: configuration files
- `docs/`: project documentation
- `scripts/`: utility and run scripts
- `models/`: model assets
- `data/`: data assets

## Run

Use the project root as working directory:

```bash
python main.py
```
